Terraform Associate 003 Certification Path

This guide outlines your journey through Terraform Academy’s interactive labs and learning modules aligned with the official HashiCorp Associate 003 exam objectives.

Phase 1: Foundations & Core Concepts

  • Terraform – Getting Started
  • Terraform HCL 100 Q’s + CLI Labs
  • AWS EC2 Lab / Azure Resource Lab

Phase 2: Provisioning Core Resources

  • AWS Labs: EC2, VPC, IAM, RDS, Security Groups
  • Azure Labs: Resource Groups, VMs, VNets, Storage

Phase 3: Infrastructure Management & State

  • Terraform Authoring & Operations
  • CLI Quizzes

Phase 4: Modules, Reuse, and Registry

  • Resource Quizzes x10
  • Use of for_each, count, inputs, outputs

Phase 5: Security, Policy, and DevOps

  • Security Lifecycle Management
  • Terraform Azure DevOps Practice

Phase 6: Scenario-Based Practice

  • Terraform Associate Practice
  • Terraform Associate 003

Phase 7: Deep Dives from HashiCorp

  • Using Terraform with AWS, Azure, Google
  • Cloud lifecycle & infrastructure architecture

Phase 8: Interactive Learning Games

  • Matchup Games: Terraform, GitOps, Azure DevOps, AWS CLF-C02, Google Cloud
  • Sequence Games: Deploy EC2 (GitOps), Build IAM policies, Create VPC, Setup Pipelines
  • Real-time score and timer feedback to reinforce understanding
  • Games are mobile-optimized and integrated into Terraform Academy
